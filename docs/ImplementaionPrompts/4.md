You are a senior .NET 8 backend developer.

Generate production-ready code for a YouTube upload queue feature.

Tech stack:

* .NET 8
* ASP.NET Core Web API
* Entity Framework Core
* PostgreSQL
* Clean architecture style
* Nullable reference types enabled

Goal:
Create a queue table that stores videos waiting to be uploaded to YouTube.

The queue worker will periodically fetch records with status `Pending` and `ScheduledUploadAt <= now`, upload the video, then update the status.

---

### Database Table

Table name: `youtube_upload_queue`

Fields:

* Id (Guid, primary key)
* Status (enum: Pending, Uploading, Uploaded, Failed)
* Priority (int, default 0)

Video metadata:

* Title (string, required)
* Description (string?)
* Tags (string[] stored as PostgreSQL text[])

Category:

* CategoryId (int, default 10)

Files:

* VideoFilePath (string, required)
* ThumbnailFilePath (string?)

YouTube result:

* YoutubeVideoId (string?)
* YoutubeUrl (string?)

Scheduling:

* ScheduledUploadAt (DateTime?) null by default 

Retry handling:

* Attempts (int, default 0)
* MaxAttempts (int, default 5)
* LastError (string?)

Audit:

* CreatedAt (DateTime, default utc now)
* UpdatedAt (DateTime)
